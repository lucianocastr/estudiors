## 1. Instalación

Descargar **WhatsApp Business** desde:
- [App Store (iPhone)](https://apps.apple.com/app/whatsapp-business/id1386412985)
- [Google Play (Android)](https://play.google.com/store/apps/details?id=com.whatsapp.w4b)

> Es una app separada de WhatsApp personal. Se pueden tener ambas en el mismo celular usando números diferentes, o migrar el número actual a WhatsApp Business (los chats y contactos se conservan).

## 3. Mensaje de bienvenida

Se envía automáticamente cuando alguien escribe por primera vez o después de 14 días sin actividad.

**Ruta:** ⚙️ Configuración → Herramientas para la empresa → Mensaje de bienvenida → Activar

**Texto:**
```
Hola 👋 Soy Romina Sanchez, abogada.

Gracias por escribirme. En breve te respondo personalmente.

Mientras tanto, si querés hacer una consulta formal podés completar el formulario en:
👉 https://www.rsestudiojuridico.com.ar/consulta

Funciona las 24hs y la consulta inicial es sin cargo.
```

---

## 4. Mensaje de ausencia

Se envía automáticamente cuando alguien escribe fuera del horario configurado.

**Ruta:** ⚙️ Configuración → Herramientas para la empresa → Mensaje de ausencia → Activar

Configurar el horario de ausencia para que coincida con fuera del horario laboral y fines de semana.

**Texto:**
```
Hola, en este momento estoy fuera del horario de atención.

Te respondo el próximo día hábil en horario de 9 a 18hs.

Si tu consulta es urgente, podés dejar tu mensaje y te contacto a la brevedad.

También podés iniciar tu consulta en cualquier momento desde:
👉 https://www.rsestudiojuridico.com.ar/consulta
```

## 5. Respuestas rápidas

Las respuestas rápidas se activan escribiendo `/` seguido de la palabra clave en cualquier chat. Evitan escribir el mismo texto repetidamente.

**Ruta:** ⚙️ Configuración → Herramientas para la empresa → Respuestas rápidas → `+`

Crear las siguientes:

---

### `/turno`
**Atajo:** `turno`
```
Para coordinar un turno, completá el formulario en:
👉 https://www.rsestudiojuridico.com.ar/consulta

Podés elegir modalidad presencial (Alta Gracia) o virtual (videollamada).
La consulta inicial es sin cargo.
```

---

### `/honorarios`
**Atajo:** `honorarios`
```
Los honorarios varían según el tipo de caso y la complejidad.

Te los informo con precisión una vez que revisemos tu situación en una primera consulta, que es sin cargo.
```

---

### `/virtual`
**Atajo:** `virtual`
```
Sí, atiendo de forma virtual por videollamada para toda la provincia de Córdoba.

Podés solicitar turno virtual en:
👉 https://www.rsestudiojuridico.com.ar/consulta
```

---

### `/presencial`
**Atajo:** `presencial`
```
El estudio está en Alta Gracia, Córdoba.

Podés solicitar turno presencial en:
👉 https://www.rsestudiojuridico.com.ar/consulta
```

---

### `/info`
**Atajo:** `info`

## 6. Etiquetas de conversación

Las etiquetas permiten organizar y filtrar conversaciones por estado.

**Cómo asignar:** Tocar y mantener presionada una conversación → Etiquetar

**Ruta para gestionar:** ⚙️ Configuración → Herramientas para la empresa → Etiquetas

Crear las siguientes etiquetas con estos colores:

| Etiqueta | Color sugerido | Cuándo usarla |
|----------|---------------|---------------|
| `Nueva consulta` | 🔵 Azul | Primer contacto sin responder todavía |
| `En seguimiento` | 🟡 Amarillo | Respondido, esperando más datos del cliente |
| `Turno coordinado` | 🟢 Verde | Tiene turno confirmado |
| `Urgente` | 🔴 Rojo | Requiere atención inmediata |
| `Cerrado` | ⚫ Gris | Caso concluido o sin respuesta del cliente |

## 7. Catálogo (opcional)

El catálogo permite crear fichas de las especialidades del estudio. Aparecen en el perfil público y se pueden compartir en los chats.

**Ruta:** ⚙️ Configuración → Herramientas para la empresa → Catálogo → `+`

Crear un ítem por especialidad:
- **Nombre:** Ej. `Derecho de Familia`
- **Descripción:** descripción corta del área
- **Link:** URL de la especialidad en el sitio (ej: `/especialidades/familia`)

---

## 8. Flujo de trabajo sugerido

```
Cliente escribe por primera vez
    → Mensaje de bienvenida automático
    → Romina recibe notificación
    → Etiqueta como "Nueva consulta"

Romina revisa y responde
    → Usa /turno, /info, /virtual según el caso
    → Si avanza → etiqueta "En seguimiento"

Se coordina turno
    → Etiqueta "Turno coordinado"
    → Cargar manualmente en el panel CRM:
       /panel/reestructuracion/nuevo (caso CRP)
       o tomar nota de la consulta

Caso cerrado
    → Etiqueta "Cerrado"
```

## Notas importantes

- **Reemplazar `https://www.rsestudiojuridico.com.ar`** con la URL real del sitio en todos los textos antes de activar.
- **Foto de perfil:** usar una foto profesional o el logo RS del estudio.
- **El número de WhatsApp Business** es el mismo que aparece en el sitio (`+54 9 351 310-0760`). Una vez migrado, los contactos existentes no notan la diferencia.
- **Estadísticas:** WhatsApp Business tiene métricas básicas de mensajes enviados/recibidos/leídos en ⚙️ → Herramientas → Estadísticas.
